# STW Metabolic Cost Classification

This repository contains the analysis code for the manuscript on sit-to-walk metabolic cost classification.

## Project description

This study used OpenSim-based musculoskeletal simulation and machine-learning models to classify simulation-derived metabolic cost levels during the sit-to-walk transition.

## Contents

- Data preprocessing
- Feature extraction
- Model training and evaluation
- SHAP-based model interpretation
- Figure generation

## Requirements

The code was developed using Python. Required packages include:

- numpy
- pandas
- scipy
- scikit-learn
- matplotlib
- seaborn
- shap
- xgboost

## Data availability

The raw experimental data are not publicly available due to participant privacy and ethical restrictions. Example or anonymized data may be provided to demonstrate the code structure.
